# CainCamera 介绍
CainCamera是一个美颜类相机，主要用于学习如何实现实时美颜、滤镜、贴纸、拍照、短视频分段录制、回删等美颜类相机预览拍照相关功能，目前动态贴纸美型美妆仍在开发当中
